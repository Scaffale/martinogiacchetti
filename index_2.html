The Office gif bot

In some Telegram chats with friends it urged the need of having some specific scene of The Office. From this need I created this bot.

It runs on Ruby on Rails, on a small dedicated server.

When queried the server searches through all the subtitles for a match. If found the scene is cutted with FFMPEG and displayed as animated gif (or muted video).

PaperUI

Sketch your own interface on paper and make it work using a Leap Motion.

Draw interface widgets on paper and link them to the widgets on your application. PaperUI interfaces through a music player, thus available mappable widgets include Play/Pause and Stop buttons as well as Volume and Seek sliders.

We worked on this project as a team of 4 people, I was responsible for the system design and the UI implementation.
